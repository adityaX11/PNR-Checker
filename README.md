# 🚆 Indian Railway PNR Status Checker

A professional Railway PNR Status Checker built using **Python**, **Streamlit**, and **RapidAPI**.

---

## Features

- 🚆 Live PNR Status
- 👥 Passenger Details
- 💰 Booking Fare
- 📅 Journey Information
- 🎯 Modern UI
- 📊 Passenger Table
- 🌙 Dark Theme
- ⚡ Fast Response
- 📱 Responsive Layout

---

## Tech Stack

- Python
- Streamlit
- Requests
- Pandas
- RapidAPI

---

## Installation

```bash
git clone https://github.com/yourusername/PNR-Status-Checker.git

cd PNR-Status-Checker

pip install -r requirements.txt

streamlit run app.py
```
